# Afterimage Website

Static GitHub Pages site for **Afterimage**, an audiovisual archive that transforms
paintings into electronic frequencies, poetic fragments, and animated memory signals.

## Files

- `index.html` - page content
- `styles.css` - visual system
- `script.js` - small scroll interaction
- `sitemap.xml` - canonical sitemap for search engines
- `robots.txt` - crawler rules and sitemap declaration

The full Starry Signal video is intentionally not committed here because the master
file is large. Host the video externally and replace the placeholder in `index.html`
when the public link is ready.
